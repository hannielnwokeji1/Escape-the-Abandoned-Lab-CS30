**Escape the Abandoned Lab**
A text-based adventure game where the player must survive and escape a dangerous experimental facility

**Features**
- Menu-driven gameplay
- Inventory system
- Health and damage system
- Random events
- Multiple endings
- Decision-based storytelling

**How to Play**
Run the python file and choose options from the menu to progress through the story

**Objective**
Escape the abandoned lab alive or expose its secrets to the world
